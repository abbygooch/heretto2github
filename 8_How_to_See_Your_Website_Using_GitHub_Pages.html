How to View Your Website Using GitHub Pages

The last step is to view your files as a website using GitHub Pages! This task will guide you through that process.

  1. Within your project repository on GitHub, click the "Settings" tab on the top of the screen.
  2. In the "Code and automation" section, click "Pages".
  3. Under "Build and Deployment", ensure "Deploy from a branch" is selected in the Source drop down box.
  4. Select your branch from the drop down menu.
    In most cases, your branch will be called "main". Your branch name is shown on the "Code" page of your GitHub repository.
  5. Click the "Save" button to the right of the branch selection drop down.
  6. Once saved, your GitHub Pages site will begin being built from your branch. Wait a few minutes while this happens.
  7. Once your site is loaded, a link will be displayed at the top of the Pages page.
  8. Click the link and you will be directed to your GitHub Pages website!
You've created a website with a shareable link!
