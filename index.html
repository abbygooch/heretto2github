Introduction: Combining Heretto and Github Pages

Welcome to our How-To Guide! This guide provides specific instructions for creating content on Heretto to be uploaded to GitHub Pages. This guide is by Abby Gooch, Amy Priester, and Natalie Briggs.

Content management can get tricky when combining multiple platforms like GitHub and Heretto. However, the collaboration of these platforms allows for a more effective use of the DITA standard. This guide offers step-by-step instructions for how to create content on Heretto and manually upload it to GitHub pages. Good luck!
